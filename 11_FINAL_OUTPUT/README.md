# 11_FINAL_OUTPUT

Cartella generata e gestita dalla pipeline v3.3 (Ciuccio Upgrade).

- **geotiff_final/**   → GeoTIFF storici georeferenziati EPSG:25832
- **vector/**          → Layer vettoriali storici (mura, strade, edifici...)
- **residual_maps/**   → Mappe residuali GCP
- **reports/**         → Report di export e validazione
- **logs/**            → Log di esecuzione pipeline

Non modificare manualmente i file qui dentro se non sai cosa stai facendo.
